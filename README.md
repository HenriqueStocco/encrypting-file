# Encrypting and Decrypting file with Python3

Clone this project
```bash
$ git clone <repository>
```

Create a virtual enviroment and activate
```
$ python3 -m venv venv
```

> Linux activate
```terminal
$ /venv/bin/activate
```
> Windows activate
```terminal
$ /venv/Scrips/activate
```
Install requirements
```terminal
$ pip install -r requirements.txt
```
